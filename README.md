# Azure IAM Home Lab

A hands-on Identity and Access Management lab built in Microsoft Azure Entra ID to demonstrate real-world IAM skills.

## What I Built

### Phase 1 — Foundation
- Created an Entra ID tenant with test users and security groups
- Assigned built-in RBAC roles (Reader) and verified least-privilege access
- Built a custom RBAC role (Lab-ResourceGroup-Reader) scoped to subscription level

### Phase 2 — Zero Trust (In Progress)
- Privileged Identity Management (PIM) with just-in-time access
- Conditional Access policies enforcing MFA
- Time-bound role activation with approval workflows

### Phase 3 — Governance (comming soon)
- Access Reviews with auto-remediation
- Entitlement Management and access packages
- Azure Policy enforcement

## Technologies Used
- Microsoft Azure Entra ID
- Azure RBAC
- Privileged Identity Management (PIM)
- Conditional Access
- Azure Policy


## Resume
This lab was built to demonstrate hands-on Azure IAM skills for Identity and Access Management roles.
