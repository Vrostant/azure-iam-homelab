# Azure IAM Home Lab

I'm building this lab to break into Identity and Access Management. 
No bootcamp. No degree. Just hands-on experience and a goal to work 
in cloud security.
This is a living project — I update it every week as I learn new things.

## Phase 1 — Foundation (Complete )

I started from zero and built this in one night.

- Spun up a Microsoft Entra ID tenant from scratch
- Created test users (Alice and Bob) and a security group called Lab-Admins
- Gave Alice the Reader role on the subscription and logged in as her to verify 
  she could see resources but not create them — least privilege in action
- Wrote a custom RBAC role from scratch using JSON called Lab-ResourceGroup-Reader
